# Cloudflare Pages environment variables

Add these variables in Cloudflare Pages project settings before deployment:

- `VITE_SUPABASE_URL`: use the Supabase project URL for project `eiubgsqvlsddvnwolfyf`.
- `VITE_SUPABASE_ANON_KEY`: use the project's public anon key.

Do not commit or expose any service-role or secret key.
