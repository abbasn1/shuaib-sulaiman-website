# Shuaib Sulaiman & Co Website

Complete React/Vite multipage trade and export website.

## Included
- Home, About, Products, Services and Contact pages
- 20 separate product-detail pages under `/export-product/:slug`
- Uploaded product images in `public/images/products`
- Responsive layout and Vercel SPA rewrite

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
